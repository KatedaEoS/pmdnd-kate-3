## 构建

如果你没有 Node.js，你可以先在[这里](https://nodejs.org/en/download)下载。

建议使用 Node.js 22 版本。

### 安装依赖

```bash
$ npm install
```

### 运行程序

```bash
$ npm run dev
```

### 运行网页端

```bash
$ npm run dev:web
```

### 编译成应用程序

```bash
# Windows
$ npm run build:win

# macOS
$ npm run build:mac

# Linux
$ npm run build:linux
```

### 编译网页端

```bash
$ npm run build:web
```
